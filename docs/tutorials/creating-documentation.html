ドキュメントの生成¶

本節では Sphinx を利用してドキュメント生成を行う方法と、そのドキュメントを無償の Read The Docs においてホストする方法について説明します。

Sphinx のインストール¶

pip を用いて Sphinx をインストールします。

pip install -U sphinx

上記以外のインストール方法については、Sphinx の installation guide を参照してください。

Sphinx をはじめよう¶

プロジェクト内に docs ディレクトリを生成して、そこにドキュメントを置きます。

cd /path/to/project
mkdir docs

その docs ディレクトリ内において sphinx-quickstart を実行します。

cd docs
sphinx-quickstart

これによりソースディレクトリが用意されて、基本的な設定内容がいくつか確認されます。 そして index.rst ファイルと conf.py ファイルが生成されます。

プロジェクトに関する情報を index.rst に記述した上で、これをビルドしてみます。

make html

ビルド操作の詳細については Read The Docs の guide を参照してください。

その他の情報¶

For a more detailed guide on how to use Sphinx and reStructuredText, please see this documentation tutorial on Hitchhiker's Guide to Python.
